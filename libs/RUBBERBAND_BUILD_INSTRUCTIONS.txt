Rubberband Library Build Instructions for Windows

1. Download Rubberband source code from https://breakfastquay.com/rubberband/
2. Extract to a temporary directory
3. Install CMake and a C++ compiler (MSVC recommended)
4. Build with CMake:

mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release -DBUILD_SHARED_LIBS=ON -DWITH_VAMP=OFF -DWITH_LADSPA=OFF -DWITH_FFTW=OFF -DWITH_KISSFFT=ON ..
cmake --build . --config Release

5. The resulting rubberband.dll will be in the build directory
6. Copy rubberband.dll to F:\CodeCamp\podium-windows\libs\

Note: Using KissFFT instead of FFTW avoids GPL linking issues and keeps the DLL self-contained (~500KB).
